# ⚔ 金雳的冒险者卡牌

> 生命领域牧师 · 第5级 · 丘陵矮人  
> 一张运行在浏览器里的 D&D 5e 人物卡，纯原生 HTML / CSS / JS，无依赖，无构建步骤。

---

## 功能概览

| 标签页 | 内容 |
|--------|------|
| ⚔ **战斗** | 角色基础信息、六维属性、HP / AC / 豁免骰、死亡豁免、主动权追踪、增益状态 |
| ☩ **职业** | 神术施法、引导神明（次数追踪）、专长列表、职业特性 |
| ✦ **法术** | 戏法 & 已准备法术（按环级分组，双列布局）、内联展开详情（金色流光边框动效）、法术选择器、一键全收起 |
| ⊛ **背包** | 物品预设（含数量 ±）、记事本、货币（铂 / 金 / 银 / 铜） |

### 其他特性
- **持久化**：所有数据保存于 `localStorage`，刷新不丢失
- **标签页记忆**：上次浏览的标签页在刷新后自动恢复
- **物品模态框**：分类筛选 + 模糊搜索，快速找到物品
- **法术详情**：金色旋转边框动效，多个详情可同时展开

## 快速开始

无需安装，直接用浏览器打开即可：

```bash
# 克隆后直接打开
git clone <repo-url>
cd DND
# 用浏览器打开 index.html
start index.html       # Windows
open index.html        # macOS
```

或者直接双击 `index.html`。

## 技术栈

- **纯原生**：零框架、零依赖、零构建
- **CSS 自定义属性**：统一深色主题色板（`--bg-deep` / `--accent-glow` 等）
- **localStorage**：轻量持久化，键名前缀 `dnd_`
- **CSS `@property` + `conic-gradient`**：法术详情金色旋转边框动效
- **字体**：[Cinzel](https://fonts.google.com/specimen/Cinzel)（标题） + 系统无衬线字体（正文）
